- Amnios : une membrane entoure l’embryon et forme un sac renfermant le liquide dans lequel le jeune se développe ( fig. 1 ; all : allantoïde, ci : chorion, cci : cavité chorionique, sv : sac vitellín). Cette membrane est l’amnios ( an ), le liquide est dit liquide amniotique ( li ). Les amniotes sont en effet des tétrapodes affranchis du milieu aquatique pour leur reproduction. L'embryon ( em ), protégé par l'œuf à coquille dure, se développe en milieu aqueux à l'intérieur de l’amnios. Chez les mammifères thériens, le développement est in utero, mais l’amnios persiste.
