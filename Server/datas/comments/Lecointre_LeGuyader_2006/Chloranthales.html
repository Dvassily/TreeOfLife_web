- Les feuilles sont opposées , dentées ; elles sont également stipulées (mais il y a beaucoup de pertes secondaires des stipules dans la famille). Les stipules sont des pièces foliaires réduites, situés de part et d’autre du pétiole.

- Les fleurs sont très petites, terminales, souvent dépourvues de périanthe . La ﬂeur d' Ardisia japonica (Chloranthacée) présente un ovaire sur lequel est fixée une étamine ( fig. 1 ; et : étamine, ova : ovaire).

- Les carpelles , libres dans les premiers stades embryologiques, sont fermés secondairement par sécrétion. Ce caractère correspond à une réversion.

- L’ovule est orthotrope , droit, c'est-à-dire que le micropyle, le hile et la chalaze sont alignés suivant l’axe de l’ovule (caractère ancestral: ovule anatrope).

- La fig. 2 présente les principales formes des ovules (ATTENTION : ce sont des schémas explicatifs, sur lesquels la vascularisation a été dessinée; elle n'est pas tout le temps présente): orthotrope ( 2a ), anatrope ( 2b ), campylotrope ( 2c ). Le sac embryonnaire ( se ) est plongé dans le nucelle ( nuc ), lui-même entouré de deux téguments, interne ( ti ) et externe ( te ). Le tégument interne se sépare du nucelle à la chalaze ( ch ), repérable sur les schémas par la vascularisation ( vasc ). Les téguments ménagent le micropyle ( mic ), par où le tube pollinique entrera. Le funicule ( f ) attache l’ovule au placenta de l’ovaire. Il se raccorde à l’ovule au hile ( h ). Dans le cas de l'ovule anatrope, le funicule est soudé à l'ovule qui a basculé sur une partie appelée le raphé ( rap ).

- Les cellules antipodiales du sac embryonnaire prolifèrent .
