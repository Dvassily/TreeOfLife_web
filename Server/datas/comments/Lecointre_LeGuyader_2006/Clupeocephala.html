- Des plaques dentaires sont fusionnées avec les éléments squelettiques les plus hauts des arcs branchiaux (squelette supportant les branchies). Les pièces concernées sont les os pharyngobranchiaux. Ici sont présentés en vue ventrale les éléments squelettiques les plus dorsaux de la corbeille branchiale (moitié droite) de l’aulopiforme Aulopus ( fig. 1 ; e1-e4 : épibranchiaux 1 à 4, mrd : muscle rétracteur dorsal, pb1-pb4 : pharyngobranchiaux 1 à 4, pd2-pd5 : plaques dentigères 2 à 5). Les pharyngobranchiaux ( pb2-pb4 ) portent des plaques dentigères ( pd2-pd5 ).

- Les gènes nucléaires codant: les ARNr 28S et 18S, rhodopsine, MLL et, plus récemment, RAG1, FEM1, PAK6, RAB15, Frizzled3, CENP-A, CutA, Gyl-t, HMG-20A, UNK-1, UNK-2, ainsi que l'analyse phylogénétique du génome mitochondrial (séquences protéiques et d’ARNt) attestent ce clade.

- Le rétro-articulaire est exclu de l’articulation carré-mandibule . La condition primitive est représentée fig.2a sur les faces internes de l’hémi-mandibule de l’ostéoglossomorphe Pantodon buchholzi ( aar  : angulo-articulaire, d : dentaire, re : rétro-articulaire; la ﬂèche montre le point d'articulation de la mandibule avec l'os carré) où le rétro-articulaire participe à l’articulation; la condition dérivée ( 2b ) est présentée chez l’aulopiforme Aulopus japonicus et chez un salmonidé ( Salmo irideus , 2c ; aar : angulo-articulaire, ca : carré, de  : dentaire, hym : hyomandibulaire, mtp : métaptérygoïde, n : neurocrâne, re : retro-articulaire, sym : symplectique) afin de situer l’articulation mandibulaire de type angulo-articulaire-carré.

- Absence d'arc neural sur le centre ural 1 . La condition primitive est présentée fig. 3 chez le fossile jurassique Leptolepides sprattiformis ( 3a , anu1 : arc neural du centre ural 1, hyp1-hyp3 : hypuraux 1 à 3), pu1 : centre pré-ural 1, pu2 : centre pré-ural2, u1 : centre ural 1, u2 : centre ural 2 et chez l’élopomorphe actuel Elops saurus ( 3b ; hyp1-hyp7 : hypuraux 1 à7, ur1-ur3  : uroneuraux 1 à 3. L’état dérivé est l'absence de cet arc neural, par exemple dans le squelette caudal du clupéomorphe Clupea pilchardus ( 3c ).

- Pas plus de six os hypuraux . Chez l’élopomorphe Elops saurus , l`état primitif présente 7 hypuraux ( 3b . hyp1-hyp7 ) ; chez le fossile Leptolepis coryphaenoides ( 3d ), on compte 9 hypuraux ( hyp1-hyp9 ) et, chez le fossile Pholidophorus bechei , on en compte 13. L’état dérivé est présenté chez le clupéomorphe Clupea pilchardus ( 3c ) avec 6 hypuraux.
