- Deux ouvertures temporales caractéristiques sur le côté du crâne: le foramen temporal ( fig. 1 ; ft ) est bordé du frontal ( fr ) et du pariétal ( pa ), le foramen auditif ( fa ) par les prootique ( pro ), exoccipital ( exo ) et basi-occipital ( bocc ).

- La vésicule antérieure est logée dans une bulle du prootique ( fig.1 , vue latérale du crâne de Dorosoma cepedianum ; bocc : basi-occipital, bpr : bulle prootique, bpt : bulle ptérotique, epo : épiotique, exo : exoccipital, fa : foramen auditif, fr : frontal, ft : foramen temporal, osp : orbitosphénoïde, pa : pariétal, pasp : parasphénoïde, pro : prootique, pto : ptérotique, rl : recessus lateralis, so : supra-occipital, spo : sphénotique), la vésicule postérieure ( bpt ) dans une bulle du ptérotique ( pto ). Le recessus lateralis ( rl ), véritable signature phylogénétique des clupéomorphes, est cette chambre dans l’os ptérotique ( pto ) où débouchent aussi plusieurs lignes latérales crâniennes. Les vésicules s’appliquent étroitement sur une fenêtre fermée seulement par une membrane fibreuse derrière laquelle se trouve la périlymphe du labyrinthe osseux de l'oreille. Ce système communiquant la pression de la vessie à l’oreille a une fonction sensorielle barométrique et hydrostatique. Il existe d’autres types de connexion directe entre la vessie natatoire et l’oreille chez certains ostéoglossomorphes ( Notopteroidei ), chez certains acanthomorphes holocentridés, et une connexion indirecte chez les ostariophyses, mais toutes reposent sur des bases anatomiques différentes.

- Connexion directe spécialisée entre l'oreille interne et la vessie natatoire . La vessie natatoire ( fig. 2 ; vn ) est prolongée en avant par un diverticule impair qui se divise en deux à son arrivée au crâne. Chacune de ces deux branches longe la région occipitale et se divise à son tour en deux branches terminées par des vésicules.
