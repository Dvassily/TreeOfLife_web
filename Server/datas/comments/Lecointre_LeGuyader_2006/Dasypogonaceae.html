- Il y a deux îlots périphériques de phloème dans les faisceaux foliaires.

- On observe un unique ovule ascendant par carpelle. La fig. 1 schématise un ovule apotrope ascendant. Si on imagine le funicule horizontal, le raphé est au-dessus de l’ovule (cas inverse: ovule épitrope ). Dans le cas précis, le funicule est situé sous l’ovule, qui est dit ascendant (cas inverse: descendant) ( ch  : chalaze, fun : funicule, mic : micropyle, nuc : nucelle, plac : placenta, rap : raphé, se : sac embryonnaire, te : tégument externe, ti : tégument interne).
