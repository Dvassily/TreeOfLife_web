- Le squelette est formé de deux composants : des spicules siliceux présentant de une (monactine) à quatre (tétractine) pointe(s) ( fig. 1 : spicules à 1 et 4 pointes) et de spongine une forme particulière de collagène , dispersée dans le parenchyme, ou structurée en larges fibres.

- Il n'y a jamais les structures simples ascon et sycon ; on ne trouve que des structures de type leucon ( ﬁg. 2a : type ascon ; b : type sycon ; c  : type leucon ; at : atrium, cch : chambre choanocytaire).
