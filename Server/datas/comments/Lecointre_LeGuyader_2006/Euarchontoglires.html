• Il existe une délétion de 6 paires de bases dans le gène de la protéine prion (gène «PRNB»).
  • Il existe une délétion caractéristique de 18 acides aminés dans la protéine SCA1 (pour « spinocerebellar ataxia type 1»).
  • Il existe une délétion caractéristique de 54 paires de bases dans le gène ATXN1 (gène de l’ataxine 1).
  • Il existe une délétion caractéristique de 3 paires de bases dans le gène «TNF» (gène du facteur de nécrose tumorale).
  • Il existe trois sites caractéristiques d’insertion d'éléments transposables de type «mariner» (LINE MLT1A0) dans le gène CFTR (pour «Cysctic Fibrosis Transmembrane Conductance Regulator», gène qui, lorsqu’il est altéré, provoque la mucoviscidose).
  • La placentation est très majoritairement hémochoriale . Dans un placenta hémochorial ( fig. 1b ), l’épithélium utérin disparaît et les capillaires utérins se distendent et forment des lacunes sanguines maternelles qui s'insèrent dans le tissu fœtal (dans le syncytium chorial). La fig. 1a est présentée pour comparaison la placentation épithéliochoriale, où l’épithélium utérin est présent.

REMARQUES :

  • Les exceptions sont les suivantes: chez les primates strepsirrhiniens (lémurs, loris), la placentation est épithéliochoriale ( fig. 1a ) et chez les scandentiens (toupayes), la placentation est endothéliochoriale, c'est-à-dire que l’épithélium utérin disparaît, et les capillaires utérins s’insèrent bien dans le tissu fœtal, mais sans former de lacunes sanguines.
  • La placentation hémochoriale se retrouve par convergence chez les fourmiliers, les tatous, les hyracoïdes, les macroscélides, les afrosoricides, certains chiroptères, certains carnivores, certains eulipotyphles, ce qui en fait un trait très labile et donc de peu de valeur phylogénétique.
  • Les phylogénies moléculaires fondées sur les séquences de 19 gènes nucléaires et 3 gènes mitochondriaux, parmi lesquels les gènes d’hémoglobines, de myoglobines, d’alpha-cristallines, de cytochrome c, de ribonucléases, etc., de gènes ribosomiques mitochondriaux regroupent tous les euarchontoglires. En outre, des analyses indépendantes d'autres gènes nucléaires (un fragment différent de RAG1, oncogène c-myc, hormone de croissance) et mitochondriaux (ND6) confirment ce clade.
