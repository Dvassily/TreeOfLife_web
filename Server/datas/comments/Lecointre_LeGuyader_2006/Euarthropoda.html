- Exosquelette  : l’animal métamérisé porte un exosquelette subdivisé en pièces, les sclérites (tergites, pleurites et sternites), articulées entre elles ( ﬁg. 1 : coupe transversale d’un segment d’arthropode; cne : cordon nerveux, cur : coeur, mus : muscles, pleu : pleurite, ster : sternite, ter : tergite, td : tube digestif).

- Appendices articulés : chaque segment porte fondamentalement une paire d’appendices articulés, ancestralement à fonction locomotrice (fig. 1).

- Les euarthropodes ont au moins une paire d’ yeux composés latéraux formés d’unités photoréceptrices indépendantes, les ommatidies ( fig. 2 : ccri : cône cristallin, cro : cornée, fac : facette, ﬁn : ﬁbre nerveuse, oma : ommatidie, rha : rhabdomère).

- Les cellules sont dépourvues de cils ou de ﬂagelles , à l’exception des spermatozoïdes de certains groupes.
