- Le calice est persistant sur le fruit. Le fruit mûr de l’herbe à Robert Geranium robertianum (Géraniacée) présente une forme caractéristique. C’est une capsule septifrage qui s'ouvre de bas en haut, isolant cinq coques contenant chacune une graine. La colonne en place est formée par les placentas ( fig.1 ; coq : coque, plac : placentas, sep : sépale, stig : stigmate, t : tige).

- Les plantes synthétisent de l’ acide ellagique .
