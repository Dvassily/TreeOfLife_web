- La bipédie implique un glissement du trou occipital ( foramen magnum ) sous la calotte crânienne ( fig. 1a : homme; 1b : gorille), une colonne vertébrale en S, ainsi que la présence de fesses.

- Le bassin ( fig. 2 ) est élargi et court chez les hominines ( 2a ), tandis qu'il est étroit et long chez les autres hominoïdes ( 2b : gorille).

- Le prémaxillaire ( fig. 3 ) présente un redressement vertical ( 3b ), tandis qu’il est oblique chez le chimpanzé ( 3a ).

- L’arcade dentaire ( fig. 4 ) est parabolique ( 4c ). L'état primitif est l'arcade en V chez les tarsiers ( 4a ), tandis qu'elle est en U chez le chimpanzé ( 4b ).
