- Gènes 18S et Hox : les phylogénies moléculaires, principalement basées sur le gène de l’ARN ribosomique 18S, ainsi que les caractéristiques de certains gènes du développement (en particulier les gènes Hox postérieurs), semblent regrouper de manière solide les animaux présentant une larve de type trochophore et les animaux possédant un lophophore (couronne de tentacules entourant la bouche), d’où le nom du groupe.
