- Feuilles : ce sont des microphylles , petites et aiguës; ﬁg. 1 : une branche de Selaginella selaginoides porte des feuilles ( ﬂe ) et des «épis» sporifères ( eps ). Elles peuvent être de grande taille chez certains fossiles.

- Sporanges : ils sont disposés sur la face supérieure ou à l’aisselle de sporophylles . Sur l'épi sporifère de Selaginella willdenovi ( ﬁg. 2a ), on remarque les macrosporophylles ( mc ) à sa base et les microsporophylles ( me ) au sommet. Les macrosporophylles ( ﬁg. 2b ) sont des feuilles qui portent à leur face supérieure un macrosporange ( mas ) contenant quatre macrospores femelles. Les microsporophylles ( ﬁg. 2c :) portent un microsporange ( mis ) contenant de très nombreuses microspores mâles.

- La ligne de déhiscence du sporange , distale ou transverse, est soulignée par des cellules pourvues d’une paroi squelettique épaissie.

- Xylème primaire : il est de type exarche ( ﬁg. 3 ), c'est-à-dire que le protoxylème ( px ) est en situation externe par rapport au métaxylème ( mxy ). Sur une coupe de la partie centrale de la tige de Lycopodium lucidulum ; le protoxylème, à petites cellules, est périphérique, tandis que le métaxylème, à grandes cellules, est central ( endo : endoderme; pec : péricycle ; phl : phloème).
