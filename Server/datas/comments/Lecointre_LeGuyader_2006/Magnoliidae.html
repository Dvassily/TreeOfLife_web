- Les feuilles présentent une marge entière.

- Les étamines sont nombreuses, spirales, extrorses , mais avec de nombreuses réversions. Ainsi, l’anthère d’une étamine de Magnolia ( fig. 1 ) comprend deux sacs polliniques ( sp ) à déhiscence longitudinale ( fd : fente de déhiscence), dans le cas présent tournés vers l’intérieur de la ﬂeur, c'est-à-dire introrses.

- Dans le sac embryonnaire, les cellules antipodes ont une vie éphémère .

- L’ovule présente une hypostase . C’est une structure formée par un groupe de cellules à paroi épaisse qui, se différenciant sous le sac embryonnaire, oblige ce dernier à se développer vers le micropyle. La fig. 2 présente un ovule anatrope avec une hypostase au niveau de la chalaze ( cn  : capuchon nucellaire, hyp : hypostase, mic : micropyle, nuc : nucelle, rap : raphé, se : sac embryonnaire, te : tégument externe, ti : tégument interne).

- Le sac embryonnaire est recouvert d’un capuchon nucellaire . L’épiderme nucellaire peut recouvrir le sac embryonnaire du côté du micropyle, formant un capuchon ( fig.3 ; cn : capuchon nucellaire, mic : micropyle, nuc : nucelle, se : sac embryonnaire, te : tégument externe, ti : tégument interne), aussi visible sur la fig. 2 .

- Le faisceau du raphé se divise à la chalaze ( fig. 4 ; alb : albumen, ch : chalaze, emb : embryon, f : funicule, fa : faisceau, mic : micropyle, nuc : nucelle, rap : raphé, te : tégument externe, ti : tégument interne).

- Les plantes synthétisent des néolignanes . Les lignanes sont des molécules polyphénoliques dérivées par dimérisation de précurseurs de la lignine, impliquant deux unités de phénylpropane et de l’acide cinnamique. Pour les néolignanes, la dimérisation se fait par les noyaux aromatiques.
