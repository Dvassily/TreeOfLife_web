- Ces animaux présentent une organisation de type méduse ; l’organisation de type polype, ancestrale, correspond à une synapomorphie du clade cnidaires. La fig. 1 résume le cycle polype/méduse d’Obelia geniculata (calyptoblastique) ; les gonophores portent les bourgeons de méduses, partie sexuée du cycle ( fec : fécondation, gon : gonophore, med : méduse, organisme sexué, per : périsarc, plan : larve planula, pol : polype, zyg : zygote).

- Les animaux peuvent avoir un squelette externe, organique, de nature chitineuse , dont le nom varie suivant les groupes. La fig. 2 représente un polype de Laodicea indica (calyptoblastique), retracté à l’intérieur de son squelette externe, ou périsarc ( per : périsarc, pol : polype, ten : tentacule) (voir aussi fig. 1 ).

- Le cnidocil , ancestralement mobile comme chez les anthozoaires, voit son appareil basal réduit et devient immobile ( fig. 3 : structure du cnidocil; cn : capsule du nématocyste, cnid : cnidocil, ster : stéréocil, sty : stylet). Le déclenchement de la décharge a lieu quand le cnidocil, rigide, se déforme et appuie sur les stéréocils.

- L’ ADN mitochondrial , ancestralement circulaire comme chez les autres eucaryotes (y compris les anthozoaires), devient linéaire .
