- Le proboscis est une trompe dévaginable séparée du tube digestif ( fig. 1 ; bo : bouche, tp :trompe).

- Le rhynchocœle ( fig. 1 : rhy ) est une cavité cœlomique tubulaire hydrostatique, entourant le proboscis. Une augmentation de pression, obtenue par contraction des muscles, entraîne la dévagination du proboscis ( a , b , c ).
