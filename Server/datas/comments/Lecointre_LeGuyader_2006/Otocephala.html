- Le pleurostyle est une baguette osseuse latérale (et donc double) rigide insérée sur le centre préural 1, et s’étendant postéro-dorsalement (une réversion chez les denticipitidés). On constate ce pleurostyle par exemple dans le squelette caudal du clupéomorphe Clupea pilchardus ( fig.1a ; hyp1-hyp6 : hypuraux 1 à 6, pl : pleurostyle, pu1 : centre pré-ural 1, pu2 : centre pré-ural 2, u1 : centre ural 1) et de l’ostariophyse Carnegiella vesca ( 1b ).

- Fusion hypural 2-centre ural 1 . La condition primitive chez l’élopomorphe Elops saurus ( fig. 2 ) montre l’hypural 2 ( hyp2 ) séparé du centre ural 1 ( u1 ) tandis que ces deux os sont réunis chez les otocéphales de la fig.1 .

- L’hypural 1 ne s’articule pas au centre ural 1 . Un espace sépare ces deux os chez les otocéphales ( fig.1 ) tandis que dans la condition primitive ils se connectent ( fig. 2 ).

- L’os palatin est un os mixte constitué d’un os enchondral, l’autopalatin, et d’un os dermique, le dermopalatin. L’autopalatin est ossifié très tôt durant l’embryogenèse chez les otocéphales (appelés aussi otomorphes ou ostarioclupéomorphes).

- La vessie natatoire se recouvre antérieurement ou complètement d’une tunique péritonéale argentée.

- Tous les gènes analysés jusqu’à ce jour montrent l’apparentement étroit entre clupéomorphes et ostariophyses, et ce avec un fort soutien: les gènes nucléaires codant: les ARNr 28S et 18S, rhodopsine, MLL et, plus récemment, RAG1, FEM1, PAK6, RAB15, Frizzled3, CENP-A, CutA, Gyl-t, HMG-20A, UNK-1, UNK-2, ainsi que l’analyse phylogénétique du génome mitochondrial (séquences protéiques et d'ARNt).

- Fusion extrascapulaire-pariétal . Le canal supratemporal passe d'un ptérotique à l’autre par les extrascapulaires. Lorsque ce canal passe par les pariétaux et le supra-occipital, on l’interprète comme le résultat d'une fusion d'un ou de plusieurs extrascapulaires avec ces éléments. Cette disposition est illustrée ici sur des vues dorsales de l’arrière du neurocrâne chez un ostariophyse anotophyse actuel Phractolaemus ansorgei ( fig.3a ; cst : canal sensoriel supratemporal, epo : épiotique, exo : exoccipital, fr : frontal, pa : pariétal, pto : ptérotique, so : supra-occipital) et chez le clupéomorphe fossile du Crétacé inférieur Spratticeps gaultinus ( fig.3b ; bocc : basi-occipital).
