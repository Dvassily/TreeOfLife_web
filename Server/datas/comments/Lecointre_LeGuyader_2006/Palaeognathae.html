• Des processus basiptérygoïdes étendus joignent les extrémités postérieures des ptérygoïdes. Sur la fig. 1 , sont représentées les vues ventrales des crânes d'un paléognathe, le nandou Rhea americana ( 1a ), d'un néognathe ansériforme, l'oie Anser anser ( 1b ) et d'un néognathe caprimulgiforme, le martinet Apus apus ( 1c ) ( bptg : basiptérygoïde, pal : palatin, poq : processus orbitaire du carré, pt : ptérygoïde, q : carré, vo  : vomer).

  • Le processus orbitaire du carré ( fig.1a ; poq ) s'articule sur le ptérygoïde ( fig.1a pt).

  • L’os dentaire est profondément fourchu dans sa partie postérieure.

  • Le calcanéum ne s'ossifie pas .

  • C’est toujours le mâle qui couve les œufs ; la femelle n'y participe pas.

  • Le groupe des paléognathes est soutenu par de nombreux caractères moléculaires issus de dix-neuf gènes nucléaires analysés conjointement ou bien séparément; dans ce dernier cas, seul Bdnf ne présente pas le regroupement.
