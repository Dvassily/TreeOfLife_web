- Chez les platyrrhiniens, les narines ( fig. 1.) s'ouvrent latéralement et sont fortement séparées ( 1b ) ; chez les catarrhiniens, elles s'ouvrent vers le bas et sont séparées par une fine cloison ( 1a ).

- Le contact entre l'os pariétal ( pa ) et l’os jugal ( j ) s'observe ( ﬁg. 2 ) chez les platyrrhiniens ( 2a : atelle). Chez un cercopithécoïde ( 2b : cercopithèque), le pariétal ne touche ni le jugal, ni l’alisphénoïde ( asp ). Chez un hominoïde ( 2c : homme), le contact est de type sphéno-pariétal a-p.

- La queue est préhensile ( ﬁg.3 ).

- L’ectotympanique est soudé à l’ouverture de la bulle tympanique, mais pas allongé en conduit auditif externe . La fig. 4 représente des coupes de la bulle tympanique telle qu’observée chez les eulipotyphles ( 4a ), les lémuriformes ( 4b ), les lorisiformes et les platyrrhiniens ( 4c ), et les tarsiiformes et les catarrhiniens ( 4d ). Les platyrrhiniens et les lorisiformes présentent de manière convergente la soudure de l’ectotympanique et du rocher. Ceci s'observe aussi chez les tarsiiformes et les catarrhiniens ( 4d ), mais l’ectotympanique s’allonge et se transforme alors en conduit auditif osseux externe.
