- Le blastopore donne fondamentalement la bouche et l'anus; la bouche est donc formée primitivement, d'où le nom du taxon ( fig. 1 : devenir du blastopore ( blp ) de l’onychophore Peripatopsis capensis , qui donne la bouche ( bo ) et l’anus ( a ), AR : arrière, AV : avant).

- Le mésoderme dérive d’un mésentoblaste .

- Le cœlome est formé par schizocœlie , c'est-à-dire que des massifs de cellules mésentoblastiques se creusent d’une cavité ( fig. 2 : larve trochophore d'annélide vue de devant, montrant le téloblaste initial donnant le mésoderme qui se creuse d’un cœlome; cœ : cœlome, mes : mésoderme, tel : téloblaste).

- Le système nerveux est ventral , à l’exception du ganglion céphalique. L’animal est de type hyponeurien, avec un collier périœsophagien ( fig. 3 : partie antérieure du système nerveux du pycnogonide Nymphon sp. ; cne : cordon nerveux, cop : collier périœsophagien, gce : ganglion cérébral, oe : œsophage).

- Exosquelette : quand il y a un squelette, il est externe.
