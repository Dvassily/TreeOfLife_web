• Le prothorax est très allongé , formant un «cou de serpent ». Le prothorax est allongé également par convergence chez certains mantoptères (mantoïdes) et chez certains névroptères (mantispidés). Fig.1 est représentée une vue dorsale d’un mâle de raphidiidé Phaerostigma notata ( pr : prothorax, sc : veine scapulaire).

  • Fait unique chez les holométaboles, le stade pupal est actif . Fig. 2 est représentée une vue dorsale d'une pupe de raphidiidé Phaerostigma notata , ( an  : antenne, fa : fourreau alaire, pa : pattes, pr : pronotom, t : tête).

  • La vénation alaire est typique : notamment, Ia veine scapulaire ( sc ) se termine sur la marge antérieure de l’aile ( fig. 1 ; sc ).
  • L’arolium du post-tarse est perdu .
