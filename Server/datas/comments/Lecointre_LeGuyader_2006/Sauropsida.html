- Il existe une quille ventrale sous les vertèbres cervicales : c’est l’hypapophyse ( hy ). Chez un synapside du Permien supérieur ( Ophiacodon , fig. 1 ), les vertèbres cervicales sont dépourvues d'hypapophyse. Les 4 vertèbres cervicales sont représentées chez un sphénodon ( fig. 2 ), chez le lézard Ophisaurus apodus ( fig. 3 ), chez le varan Varanus monitor ( ﬁg. 4 ). Sur la ﬁg. 5 sont représentées les deux premières vertèbres cervicales et une vertèbre dorsale antérieure de la couleuvre Coluber gemonensis ( arn : arc neural, cy : condyle, di : diapophyse, pap : parapophyse, ple : pleurocentre, ply : pleurapophyse, pra : proatlas, pz : postzygapophyse, Z : zygosphène).
