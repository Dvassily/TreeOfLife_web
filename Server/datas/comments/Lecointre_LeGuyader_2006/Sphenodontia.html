• Le prémaxillaire porte 2 ou 3 dents fusionnées . Un crâne de Sphenodon punctatus est représenté en vue latérale fig. 1 ( df : dents fusionnées, pmx : prémaxillaire). Ces dents fusionnées apparaissent de maniére convergente chez les rhynchosaures du Trias.

  • Dents: il existe une double rangée de dents à la mâchoire supérieure : la rangée externe de dents maxillaires, et une rangée interne de dents palatines. Un crâne de Sphenodon punctatus est représenté fig. 2 en vue ventrale ( dm : dent maxillaire, dp : dent palatine, mx : maxillaire, ni : narine interne (ou choane), pal : palatin, pg : ptérygoide, vo : vomer). Les os palatins ( pal ) sont élargis, refermant partiellement la fenêtre palatine ( fpa ), ou fenêtre sous-orbitaire.

  • Les dents sont acrodontes , c’est-à-dire qu’elles sont implantées par leur base sur la pièce squelettique par ankylose ( fig. 3 a ). Par exemple, un autre type d’implantation par ankylose est la pleurodontie ( ﬁg. 3b ), où Ie support osseux est creusé d’un sillon sur son bord externe (labial), le long duquel les dents sont ﬁxées par leur face Iatérale. Enﬁn, la dent implantée dans une alvéole est dite thécodonte ( ﬁg. 3c ). Sur la fig. 3 sont représentées des coupes transversales schématiques de mandibules d’un sphénodon ( 3a ), d’un lézard ( 3b ) et d’un crocodile ( 3c ). Les dents acrodontes sont trouvées par convergence également chez certains lézards comme les agamidés, les caméléonidés et les amphisbènes trogonophinés. La pleurodontie est trouvée chez les autres lézards (sauf les amphisbénidés), et les dents thécodontes chez les archosaures, et donc les crocodiles.
