- Tous les euarchontoglires ont une placentation hémochoriale sauf les scandentiens et les strepsirrhiniens. Les strepsirrhiniens ont une placentation épithéliochoriale, ce qui n’est pas le mode de placentation le plus fréquent chez les mammifères euthériens. Dans cette placentation ( fig. 2a ), l’épithélium utérin est présent, et les capillaires utérins ne pénètrent pas les tissus du placenta. Dans un placenta hémochorial ( fig. 2b ), l’épithélium utérin disparaît et les capillaires utérins se distendent et forment des lacunes sanguines maternelles qui s’insèrent dans le tissu fœtal (dans le syncytium chorial).

REMARQUE: la placentation épithéliochoriale se retrouve par convergence chez les fereongulés regroupant les cétartiodactyles, les périssodactyles, les pholidotes et les carnivores (ces derniers ayant acquis secondairement un autre mode de placentation).
