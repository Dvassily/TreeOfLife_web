Les synapomorphies morpho-anatomiques des nœuds basaux des Angiospermes sont très discutées. Elles portent sur la structure de la paroi du grain de pollen et sur des caractères des vaisseaux, et il n’y a actuellement pas de consensus. La phylogénie est principalement fondée sur des caractères moléculaires.

- Le bois comprend des vaisseaux dont les éléments présentent des perforations terminales scalariformes ( fig. 1 ; po : ponctuation, psc : perforation scalariforme).

- Le grain de pollen est tecté-columellé . La paroi squelettique du grain de pollen est très structurée, en plusieurs couches. La fig. 2 schématise les éléments de base ( cb : couche basale, col : columelle, elsc : éléments sculpturaux, end : endexine, int : intine, nex : nexine, tect : tectum). Dans le cas présent, la paroi squelettique présente un tectum et des columelles.

- Le tectum est réticulé , c’est-à-dire que, vu de dessus, il présente une structure réticulée, comme sur la ﬁg. 3 ( cb : couche basale, tect : tectum).

- Dans le sac embryonnaire, le noyau de l’oosphère est frère de l'un des deux noyaux polaires.

- Des motifs DEAER (codant pour: acide aspartique-acide glutamique-alanine-acide glutamique-arginine) ont été perdus dans les gènes homéotiques floraux à MADS-box AP3 ( APETALA 3 ) et PI ( PISTILLATA ).
