- Les inﬂorescences sont composées de cymes hélicoïdes ramassées, pourvues de nombreuses fleurs. Une cyme hélicoïde est une inflorescence à croissance sympodique, dont la ramification change de sens à chaque degré ( fig. 1 ; ax1 : axe 1, ax2 : axe 2, bﬂ : bourgeon floral, br : bractée, bt : bourgeon terminal).

- Le tapis staminal est invasif ou plasmodial . Les cellules sont invasives quand elles s’insinuent entre les cellules sporogènes. Le tapis est plasmodial lorsque ses cellules dégénèrent, les membranes se lysent et le contenu des cellules se déverse dans le sac pollinique pour y former un plasmodium nutritif, au sein duquel les noyaux restent fonctionnels pendant quelque temps.
