• Du point de vue moléculaire, la relation de groupe frère entre les psittaciformes et les passériformes est très bien soutenue; il s’agit là d'un résultat majeur de ces dernières années. Ce groupe, qui n’a pas encore de nom, est soutenu par de nombreux caractères moléculaires issus de dix-neuf gènes nucléaires analysés conjointement.
